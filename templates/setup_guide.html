Thermal Printer Setup Guide

Setting up your thermal printer for text printing

This guide will help you set up your thermal printer with the Bluetooth Print app for text printing tests.

1Install the Bluetooth Print app

Download and install the "Bluetooth Print" app from Google Play Store:

Download App
2Connect your printer

Connect your thermal printer to your Android device:

For Bluetooth printers:
  1. Turn on your thermal printer
  2. Go to your Android device's Bluetooth settings
  3. Make sure Bluetooth is enabled
  4. Select "Scan" or "Search for devices"
  5. Select your printer from the list of available devices
  6. Follow any pairing instructions (default PIN is often 0000 or 1234)
For USB printers:
  1. Connect your printer to your Android device using an OTG (On-The-Go) cable
  2. Your device should detect the printer automatically
  3. In the Bluetooth Print app, go to Settings and select USB connection mode
3Enable Browser Print function

Open the Bluetooth Print app and follow these steps:

  1. Tap on the menu icon (three horizontal lines) in the top left corner
  2. Select "Settings"
  3. Find and enable "Browser Print" option - it should be toggled ON
  4. Allow any permission requests from the app
  5. Select your printer from the list of available devices in the app
  6. Test the connection using the app's built-in test print function
4Configure text settings (for 58mm printer)

For best results with text formatting, set these options in the app:

  • Paper Width: Select 58mm
  • Print Density: Medium (adjust as needed for text clarity)
  • Character Set: UTF-8 for international character support
  • Text Mode: Enable rich text formatting support
5Test your setup

Use our test page to verify text formatting is working correctly:

Go to Test Page
Troubleshooting
  • Text formatting not applying: Make sure rich text formatting is enabled in app settings
  • Text too small or large: Check paper width settings (should be 58mm)
  • Printer not detected: Make sure it's powered on and properly connected
  • App does not launch when clicking Print: Check if "Browser Print" is enabled
  • No printing occurs: Verify printer selection in the app
  • Print quality issues: Adjust print density in app settings
Tips for Text Formatting
  • Keep text lines under 32 characters for 58mm printers to avoid word wrapping
  • Use centered text for titles and headings
  • Bold text works best for emphasis rather than large text
  • Some older printer models may have limited support for advanced text formatting
  • Test with different font sizes to find the optimal readability for your printer
